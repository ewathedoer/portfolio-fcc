Code & Design Portfolio

  • Simon Game (FCC project)
  • The memory game where a user has to repeat the sequence of 20 sounds in order to win. As the game progresses, the number of buttons to be pressed increases. There are two modes implemented (strict and regular).
  • Wiki Viewer (FCC project)
  • Wiki Viewer App is the project where a user can search through the extracts of Wikipedia articles. Short entries enable quick scanning and deciding which article should be read fully. Choose For Me button enables learning about something random.
  • Game Watcher (FCC project)
  • In Game Watcher a user can look for a favorite game and who's playing it now. Users can add Twitch players to the list to follow them. NOTE: sign up to Twitch to watch the streams after getting redirected there. To store user's choices local storage is implemented.
  • JavaScript Calculator (FCC project)
  • The calculator was built using Materialize framework. It allows basic math calculations.
  • Tic Tac Toe game (FCC project)
  • Tic Tac Toe game has two modes implemented: a beginner and an advanced player logic. The results are kept in score box. A user can change both mode and symbol he plays with.
  • Pomodoro Clock (FCC project)
  • Pomodoro Clock is a great tool for time management. A user can customize the time of work and break sessions, declare the goal, go social with their work progress and analyze the pomodoro history of finished cycles. Check this quick wideo tour.
  • Quote Machine (FCC project)
  • The Bootstrap website designed and implemented during FCC course according to quote machine project specification. The projects includes API and Twitter integrations. Each quote can be shared on Twitter.
  • Sheep Weather App (FCC project)
  • The weather app checks weather in current location. The UI is built on Bootstrap. I designed and implemented the idea during FCC course according to weather app project specification. The app uses API integration and geolocation.
  • Tribute (FCC project)
  • The Bootstrap website designed and implemented during FCC course according to tribute page project specification.
  • Portfolio (FCC project)
  • The website designed and implemented during FCC course according to portfolio page project specification. It is built on Bootstrap. This is the page you are currently on.
  • ProCreative website
  • The company website. I designed and implemented the whole website from scratch. The copy, cartoons, and movies are also created by me. The project is built using Polymer framework, CSS, HTML, Javascript, and jQuery.
  • Costa del Sol
  • The landing website designed and implemented serving as an introduction to my tribute to Andalucia, Spain. The page uses smooth scrolling and Google Maps integration. It is built with HTML, CSS, and Javascript.
  • Personal website
  • The personal website of a doer. The site serves as the about type of page. This is a simple page built from scratch with Javascript, HTML, and CSS.
